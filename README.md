# 家庭财务规划系统

一个功能完整的家庭财务管理与预测网站，帮助用户规划长期财务目标。

## 功能特点

### 📋 基础信息输入
- 当前年龄和预期退休年龄
- 当前资产总额
- 当前月收入和收入年增长率
- 预期投资收益率

### 🏦 房贷管理
- 房贷总额、年限和月供计算
- 自动计算剩余房贷余额

### 💰 支出管理
- 详细的家庭支出明细分类
- 支持自定义添加支出项目
- 通胀率设置和自动计算

### ⚠️ 风险模拟
- 失业年份设定
- 失业持续月数配置
- 模拟失业对财务状况的影响

### 💡 智能投资建议
- 基于财务状况的风险承受能力评估
- 个性化投资配置建议（股票、债券、基金、黄金等）
- 详细的财务健康建议

### 📊 可视化分析
- 资产增长趋势图
- 收入支出对比图
- 详细的年度财务数据表格
- 关键财务指标总结

## 使用说明

1. **填写基础信息**：输入当前年龄、资产、收入等基础数据
2. **设置房贷信息**：如果有房贷，填写相关参数
3. **管理支出明细**：添加和编辑家庭各项支出
4. **配置经济参数**：设置通胀率和预期投资回报率
5. **模拟风险场景**：可选择性设置失业等风险情况
6. **生成预测报告**：点击计算按钮，查看完整的财务预测和分析

## 计算逻辑

### 收入增长
年度收入 = 当前月收入 × 12 × (1 + 收入年增长率)^年份

### 支出通胀
年度支出 = 月度支出总额 × 12 × (1 + 通胀率)^年份

### 房贷计算
逐年递减，直到还清

### 投资收益
年投资收益 = 当前资产 × 投资收益率

### 资产变化
资产变化 = 可投资金额 + 投资收益

## 注意事项

- 所有金额单位为人民币元
- 收入增长率和通胀率按百分比输入（例如：5 表示 5%）
- 预测结果仅供参考，实际投资需谨慎
- 建议定期（每半年）重新评估和调整计划

## 快捷脚本

为了方便本地开发，项目提供了以下 Windows 批处理脚本：

| 脚本文件 | 功能描述 |
|----------|----------|
| `build.bat` | 清理并构建生产版本到 `dist/` 文件夹 |
| `preview.bat` | 启动预览服务器并打开浏览器（自动构建） |
| `build-and-preview.bat` | 一键完成清理、构建、预览和打开浏览器 |

使用方式：双击脚本文件运行，或命令行中执行。
